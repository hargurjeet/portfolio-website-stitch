# Project Product Specification: Portfolio Website

## Overview
A premium, interactive personal portfolio website showcasing Hargurjeet's professional experience, case studies/projects, technical skills, and contact pathways.

## Core Features
1. **Hero Screen**: High-impact introduction, bold typography, subheadings, and quick action buttons.
2. **Projects Showcase**: Interactive bento-grid display of featured projects and case studies, complete with tag filters and detail views.
3. **Experience Timeline**: Vertical interactive timeline detailing career trajectory and significant achievements.
4. **Skills Visualization**: Modern visual display of technical strengths and tool proficiencies.
5. **Contact Channel**: Sleek contact form or floating glassmorphic widget for visitor messages.
