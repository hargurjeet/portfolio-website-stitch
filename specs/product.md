# Project Product Specification: Bespoke Data Lead Portfolio

## Overview
A premium, interactive personal portfolio website showcasing Hargurjeet's professional experience as an AI Architect, case studies/projects, technical skills, and contact pathways.

The visual layout and interactive behaviors must strictly replicate the pre-built **Hargurjeet Portfolio - Refined Scientific Pop** design from Stitch (Screen ID: `db841db45fea4692b7951c9adae25064`).

## Core Features
1. **Header & Navigation**: Features logo `H_SINGH_AI` and links to Projects, Experience, Stack, and Contact, with a prominent "RESUME.PDF" action button.
2. **Hero Screen**: High-impact bio card with principal title `[ PRINCIPAL_GENAI_ARCHITECT ]`, body intro, and portrait card displaying greyscale profile image.
3. **Metric Card**: High-contrast block displaying "70% Reduction in Model Latency" in large Source Serif 4 typography.
4. **Technical Stack Bento**: White neo-brutalist container displaying key language and tooling chips, and lists detailing Specializations and Philosophy.
5. **Projects Grid**: Multi-card bento grid displaying featured case studies (e.g. Antigravity, RAG Evaluation, Pulse Stream) with tags and collapsible details.
6. **Experience & Academic Log**: Sequential narrative blocks detailing career chronology (BT, Shell, TCS) and educational history.
