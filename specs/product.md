# Project Product Specification: Bespoke Data Lead Portfolio

## Overview
A premium, interactive personal portfolio website showcasing Hargurjeet's professional experience as a Data Lead, case studies/projects, technical skills, and contact pathways.

The visual layout and interactive behaviors must strictly replicate the pre-built **Bespoke Data Lead Portfolio** design system from Stitch (Project ID: `projects/6624443985264640229`).

## Core Features
1. **Hero Screen**: High-impact editorial introduction featuring a bold Source Serif 4 title, Geist body subheadings, and Cyber Lime rectangular call-to-action buttons.
2. **Projects Showcase**: Interactive bento-grid display of featured projects and case studies, using flat white surfaces, heavy 2.5px borders, and monospaced label-caps tags.
3. **Experience Timeline**: Vertical structural timeline detailing career trajectory, utilizing Cyber Lime square markers.
4. **Skills Visualization**: Modern visual display of technical strengths and tool proficiencies, formatted as flat, segmented progress bars resembling retro loading bars.
5. **Contact Channel**: Sleek, high-contrast contact form matching the neo-brutalist theme.
